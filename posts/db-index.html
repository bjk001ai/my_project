📇 DB 인덱스, 왜 쓰고 어떻게 타나?

2026.05.16 · 샘플 글

인덱스는 테이블 검색을 빠르게 하는 자료구조(B-Tree 등)다. 대신 INSERT/UPDATE 시 인덱스도 같이 갱신된다.

언제 거나?

WHERE, JOIN, ORDER BY에 자주 쓰는 컬럼에 건다. 카디널리티가 너무 낮은 컬럼(성별만)은 효과가 적다.

예시 (SQL)

CREATE INDEX idx_user_email ON users(email);

실행 계획(EXPLAIN)으로 인덱스를 타는지 확인하는 습관을 들이자.
