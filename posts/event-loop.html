JS는 싱글 스레드지만, 콜 스택·태스크 큐·마이크로태스크로 비동기를 처리한다.

핵심 구성

Call Stack → Web APIs → Callback Queue → Event Loop가 스택이 비면 큐에서 꺼낸다.

예시

console.log('1');
setTimeout(() => console.log('2'), 0);
console.log('3');
// 1, 3, 2
